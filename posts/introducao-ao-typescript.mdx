# Introdução ao TypeScript

TypeScript é um superset do JavaScript que adiciona tipagem estática ao código, tornando-o mais robusto e fácil de manter.

## Por que usar TypeScript?

1. **Type Safety** - Detecta erros em tempo de desenvolvimento
2. **Melhor IDE Support** - Autocomplete e refactoring inteligente
3. **Documentação viva** - Os tipos servem como documentação
4. **Escalabilidade** - Facilita o trabalho em projetos grandes

## Exemplo básico

```typescript
interface User {
  name: string;
  email: string;
  age: number;
}

function greetUser(user: User): string {
  return `Olá, ${user.name}!`;
}
```

## Conclusão

TypeScript é uma ferramenta poderosa que pode melhorar significativamente a qualidade do seu código e a produtividade do desenvolvimento.
